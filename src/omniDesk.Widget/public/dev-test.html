OmniDesk Widget — Dev Test

Página local para validar QS-1 (instalação + envio de primeira mensagem) durante o desenvolvimento.

Ajuste OmniDeskConfig.token abaixo para apontar ao tenant de teste e rode npm run dev em src/omniDesk.Widget para servir os assets em http://localhost:5173.

Snippet de instalação

<script>
  window.OmniDeskConfig = {
    token: '00000000-0000-0000-0000-000000000000',
    apiBaseUrl: 'http://localhost:5000'
  };
</script>
<script src="http://localhost:5173/widget/v1/loader.js"></script>

Após carregar, um botão flutuante aparecerá no canto inferior. Clique para abrir, marque o checkbox LGPD e envie uma mensagem.
